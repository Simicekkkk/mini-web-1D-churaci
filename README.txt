Školní E-Sport Turnaj - Webový Projekt

Tento projekt vznikl jako školní úloha pro procvičení spolupráce v týmu pomocí Git/GitHubu a základů tvorby webu.

Autoři
Ondřej Vondraš (@VelkePeroKingVon)
Šimon Sebastian Saidl (@Simicekkkk)

Co projekt obsahuje
-Interaktivní jednostránkový web pro registraci na turnaj.
-Seznam aktivit a her.
-Responzivní registrační formulář.
-Moderní "dark-mode" design pomocí CSS Flexboxu.

Rozdělení práce
Ondřej Vondraš: Tvorba HTML struktury, texty a základní rozvržení.
Šimon Sebastian Saidl: Kompletní CSS styling, design formuláře a nastavení repozitáře.

Postup práce
1. Vytvoření repozitáře na GitHubu a pozvání spolupracovníka.
2. Sestavení základní HTML kostry s povinnými prvky.
3. Stylizování webu pomocí CSS (barvy, písmo, flexbox).
4. Zpracování a úprava registračního formuláře.
5. Dokončení README.txt a odevzdání odkazu.

Jak projekt spustit
1. Otevřete soubor "index.html" v libovolném webovém prohlížeči (Chrome, Firefox, Edge).
